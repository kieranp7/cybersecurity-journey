# Lab: Testing a Honeypot

## Objective
The objective of this lab was to take a quick look at honeypots and what they do, using Kali Linux NMAP and Valhalla on Windows XP.

## Steps taken
- Created a honeypot using Valhalla in a Windows XP VM
- Used Kali Linux terminal NMap to try and attack the XP machine to find out what happens
