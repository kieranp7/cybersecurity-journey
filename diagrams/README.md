# Diagrams

This folder holds visual material created while working through the cybersecurity journey - anything that's easier to understand as a picture than as text.

## What goes here

- Network topology diagrams (home lab builds, segmented test networks, VPN setups)
- Attack chain / kill chain diagrams for specific labs or CTFs
- Active Directory structure maps (domains, trusts, OUs, privilege paths)
- Annotated protocol or model diagrams (OSI layers, TCP handshake, auth flows)
- Architecture sketches for anything built from scratch (e.g. a home SOC, a vulnerable VM environment)

## Why it's here

Diagrams are quick evidence that I understand *how something fits together*, not just that I followed a set of steps. They're also genuinely useful to look back on - a topology diagram from three months ago is a faster refresher than re-reading old notes.

## Format

- PNG or SVG preferred for anything shared here, with the source file (e.g. `.drawio`) included alongside it where possible
- Simple naming convention: `YYYY-MM-DD_short-description.png`
- A one-line caption in this README or a commit message explaining context, where it's not obvious from the filename
